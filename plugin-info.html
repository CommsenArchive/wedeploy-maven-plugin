WeDeploy Maven Plugin

  • | Last Published: 2017-08-29
  • Version: 0.1.0-SNAPSHOT

Plugin Documentation

Goals available for this plugin:

Goal Description
we:deploy Deploys a set of files to given WeDeploy project.
we:help Display help information on wedeploy-maven-plugin.
Call mvn we:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
we:info Displays the status of WeDeploy's online services.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.commsen.wedeploy</groupId>
          <artifactId>wedeploy-maven-plugin</artifactId>
          <version>0.1.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.commsen.wedeploy</groupId>
        <artifactId>wedeploy-maven-plugin</artifactId>
        <version>0.1.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
